# Voyager

## Prerequisites
- Node.js & npm
- Vue CLI

### Recommended
- Visual Studio Code (with extensions: Vetur, TSLint, Chrome debugger)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run test:e2e
```
